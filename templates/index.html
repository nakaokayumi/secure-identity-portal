This project is a Secure Identity & Access Management (IAM) Portal built to demonstrate secure web authentication. I acted as the Lead Security Developer, focusing on protecting user data through backend logic.

I utilized Flask for the backend and implemented PBKDF2 password hashing. This ensures that even if the database is stolen, user passwords remain unreadable. The result is a functional, secure portal that tracks user logins for compliance auditing.
